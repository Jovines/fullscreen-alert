# DESIGN.md

全屏提示卡片的设计规范。配色参考 Ghostty 默认终端主题（Tomorrow Night），目标是柔和的暗色环境，文字在任何亮/暗背景下保持可读。

## 色彩体系

| 角色 | 色值 | 说明 |
|------|------|------|
| 黑 `#1D1F21` | `rgb(29, 31, 33)` | 卡片底色，带蓝灰调，非纯黑 |
| 白 `#C5C8C6` | `rgb(197, 200, 198)` | 正文色，暖灰白，非纯白 |
| 蓝 `#81A2BE` | `rgb(129, 162, 190)` | 强调色，钢蓝，替代默认蓝 |
| 灰 `#666666` | `rgb(102, 102, 102)` | 弱提示色，如底部操作提示 |

## 卡片结构

```
┌─────────────────────────────────────────┐
│ layer                α 0.95  #1D1F21   │  ← 卡片底层
│ darkOverlay          α 0.65  #1D1F21   │  ← 压暗磨砂透光
│ NSVisualEffectView   .dark / behindWindow │ ← 磨砂玻璃
│                                         │
│  标题            α 0.55  #C5C8C6       │
│  ┃ prompt       α 0.7   #C5C8C6       │  ← 左侧竖线 #81A2BE
│  WebView        #C5C8C6 / #1D1F21      │
│  路径 分支 模型  α 0.5   #C5C8C6       │
│  操作提示        #666666                │
└─────────────────────────────────────────┘
border              α 0.1   #C5C8C6      默认
border (hover)      α 0.35  #C5C8C6      高亮
```

## 窗口遮罩

全屏半透明蒙层：`NSColor.black α 0.5`
